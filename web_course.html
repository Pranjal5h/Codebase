Web Development Course

1. Introduction to Web Development

Understand what web development is, how websites work, and the roadmap to becoming a developer.

2. What is HTML?

Learn the basics of HTML, the backbone of the web, and create your first simple webpage.

3. HTML Nested Lists & Tables

Explore ordered, unordered, and nested lists along with tables to structure data.

4. File Path, Boilerplate, Div, Class, ID

Understand HTML boilerplate, file paths, and the role of div, class, and ID for better structure.

5. HTML Forms

Create interactive forms using input, labels, checkboxes, radio buttons, and text areas.

6. Deploy Your First Website

Learn how to take your project online and make it accessible on the internet.

7. What is CSS (Cascading Style Sheets)?

Discover how CSS adds styling, colors, and layouts to your webpages.

8. CSS Box Model & Display Property

Understand margins, padding, borders, and how display properties affect elements.

9. Cascade & Positioning (2 Mini Projects)

Learn positioning, layering, and the cascade effect while building two projects.

10. Flexbox in CSS (One Project)

Master Flexbox to build responsive layouts with an easy project example.

11. Mastering CSS Grid (In One Shot)

Dive into CSS Grid to design advanced, flexible, and responsive page structures.

12. CSS Media Queries & Box Shadow

Use media queries for responsive design and add creative shadows to elements.

13. CSS Animation

Bring your webpages to life with transitions, keyframes, and animations.

14. CSS Transitions & Transforms

Learn how to smoothly change element styles with transitions and use transforms to rotate, scale, skew, and move elements.
